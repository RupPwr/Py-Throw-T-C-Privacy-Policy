Privacy Policy

Py Throw

Last Updated: [12 September 2025]

Developed by: Rup's Laboratory

1. Introduction

Welcome to Py Throw, an offline Python interpreter application developed by Rup's Laboratory. This Privacy Policy explains how we handle, collect, and protect your information when you use our application. Py Throw is a specially designed educational application created exclusively for students to write, execute, and learn Python programming directly on their Android devices without requiring an internet connection.

2. Data Collection and Use

Py Throw is designed with minimal data collection in mind. The application only collects data that you explicitly provide: • User-Created Python Files: Code files and scripts that you create and save • User-Generated Content: Any text, code, or files that you upload or create Your data is used solely to enable you to write, edit, execute Python code, store your files, and provide educational functionality.

3. Data Storage and Security

Local Storage Only: All data collected by Py Throw is stored exclusively on your device's internal storage. We do not transmit, upload, or store your data on any remote servers or cloud services. No Data Transmission: Py Throw operates completely offline and does not have internet connectivity permissions. Your privacy stays with your device. Data Deletion: You have full control over your files and can delete them at any time directly through the application.

4. Permissions

Required Permissions:
• READ_MEDIA_DOCUMENTS – Access and open user-selected Python files and documents
• READ_EXTERNAL_STORAGE – Read existing user files (Android versions below 13)
• WRITE_EXTERNAL_STORAGE – Save user-created Python files (Android versions below 10)

No Internet Permission:
Py Throw does NOT request internet permission and cannot access the internet in any form.

Storage Access Justification:
Storage access is used strictly for allowing users to create, edit, save, and manage their own Python scripts locally on their device. The application does not access unrelated files or personal media.

No Package Installation:
Only built-in Python standard libraries are supported. Installation of external packages, system commands, or dynamic code downloads is not allowed.

5. Third-Party Services

No Third-Party Data Sharing: We do not share your data with any third-party services, advertisers, or analytics platforms.

Open-Source Libraries:
• Python - Python Software Foundation
• Chaquopy - Python interpreter for Android
• Kotlin Coroutines - Background operations
• JUnit - Testing

No tracking, crash reporting, or analytics integrated.

6. Advertisements

Py Throw is an ad-free application. We do not display any advertisements, sponsored content, or promotional materials. You can use the application without interruptions or commercial distractions.

7. User Accounts

Py Throw does not require user accounts, registration, or authentication. You can use the application immediately without creating an account or providing personal information such as name, email, or phone number.

8. Data Sharing

User-Initiated Sharing Only: Any sharing of files is entirely under your control. Manual Export: Share through Android's native share menu, file managers, Bluetooth, or other installed apps. The application does not perform automatic data transmission.

9. Security

Data Protection: All user data, including Python scripts and files, is stored locally on the user's device. Py Throw does not transmit, upload, or share any data externally. Code Execution Disclaimer: Py Throw allows users to write and execute Python code locally for educational and learning purposes only. Users are fully responsible for any code they create or execute. The developer does not promote or support hacking, malware creation, unauthorized access, or any illegal activities. Device Security: Users are encouraged to protect their devices using a secure lock such as a PIN, password, or biometric authentication.

10. Children's Privacy

Py Throw is not directed at children under the age of 13. The application does not knowingly collect personal information from children. If you believe that a child has provided personal data, please contact us so that appropriate action can be taken.

11. Changes to This Policy

We reserve the right to update or modify this Privacy Policy at any time. Any changes will be reflected with an updated date at the top of this policy. Your continued use constitutes acceptance of the updated policy.
