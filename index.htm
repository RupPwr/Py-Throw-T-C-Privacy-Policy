Py Throw

Last Updated: 26 December 2025

1. Introduction

Welcome to Py Throw, an offline Python interpreter application developed by Rup's Laboratory. This Privacy Policy explains how we handle, collect, and protect your information when you use our application. Py Throw is a specially designed educational application created exclusively for students to write, execute, and learn Python programming directly on their Android devices without requiring an internet connection.

2. Data Collection and Use

Py Throw is designed with minimal data collection in mind. The application only collects data that you explicitly provide:

- User-Created Python Files: Code files and scripts that you create and save
- User-Generated Content: Any text, code, or files that you upload or create

Your data is used solely to enable you to write, edit, execute Python code, store your files, and provide educational functionality. All data access occurs only after explicit user action. Py Throw does not access any files or data without user knowledge or consent.

3. File Storage and Access

Local Storage Only:
All Python (.py) files created or imported using Py Throw are stored locally on the user's device.

User-Selected Storage Location:
Py Throw allows users to select a preferred folder using Android's system file picker during the first file save. Storage access is granted only after explicit user action and consent through the system interface.

Once a folder is selected, that location is saved in the application and can be changed by the user through the app interface. The application automatically saves all files to the chosen location. Py Throw does not access any files or folders that are not explicitly selected by the user.

No Data Transmission:
Py Throw operates completely offline and does not transmit, upload, or share any user data.

4. Permissions

No Internet Permission:
Py Throw does NOT request internet permission and cannot access the internet in any form.

Storage Access:
Py Throw uses Android's system file picker to allow users to select folders for storing Python files. Storage access is granted only after user approval through the system interface during the first file save.

The application does not request broad storage permissions and does not scan or access unrelated files or personal media.

5. Third-Party Services

No Third-Party Data Sharing: We do not share your data with any third-party services, advertisers, or analytics platforms.

Open-Source Libraries:
- Python - Python Software Foundation
- Chaquopy - Python interpreter for Android
- Kotlin Coroutines - Background operations
- JUnit - Testing

No tracking, crash reporting, or analytics integrated.

6. Advertisements

Py Throw is an ad-free application. We do not display any advertisements, sponsored content, or promotional materials. You can use the application without interruptions or commercial distractions.

7. User Accounts

Py Throw does not require user accounts, registration, or authentication. You can use the application immediately without creating an account or providing personal information such as name, email, or phone number.

8. Data Sharing

User-Initiated Sharing Only: Any sharing of files is entirely under your control. Manual Export: Share through Android's native share menu, file managers, Bluetooth, or other installed apps. The application does not perform automatic data transmission or any background file access.

9. Security

Data Protection: All user data, including Python scripts and files, is stored locally on the user's device.

Code Execution Disclaimer: Py Throw allows users to write and execute Python code locally for educational and learning purposes only. Users are fully responsible for any code they create or execute. The developer does not promote or support hacking, malware creation, unauthorized access, or any illegal activities.

Device Security: Users are encouraged to protect their devices using a secure lock such as a PIN, password, or biometric authentication.

10. Children's Privacy

Py Throw is not directed at children under the age of 13. The application does not knowingly collect personal information from children. If you believe that a child has provided personal data, please contact us so that appropriate action can be taken.

11. Changes to This Policy

We reserve the right to update or modify this Privacy Policy at any time. Any changes will be reflected with an updated date at the top of this policy. Your continued use constitutes acceptance of the updated policy.
